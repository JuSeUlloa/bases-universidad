## Objetivo
Este repositorio contiene el desarrollo del taller de SCRIPTS SQL (DDL y DML) y Python de la base de datos suministada Universidades.

### 1. Descarga Plugin (extencion) Python 

![Imagen de Git](https://www.python.org/static/img/python-logo.png)

Abrir el repositorio clonado con el codigo en un editor de codigo, es necesario instalar la extencion para python y poder seleccionar el entorno de ejecucion.

[Tutorial Plugin VSCode](https://code-visualstudio-com.translate.goog/docs/languages/python?_x_tr_sl=en&_x_tr_tl=es&_x_tr_hl=es&_x_tr_pto=tc)


### 2. Instalación de entorno virual 
Ubicar la consola o terminal sobre el proyecto clonado antes de realizar la ejecuccion de cualquier comando. 

- comando para configurar el nombre de usuario:
  ```
  python -m venv bases-universidad 
  ```
### 3. Instalacion de liberias
- comando para instalar liberias requeridas para el proyecto:
  ```
  pip install sqlite3 pandas matplotlib seaborn
  ```

### 4. Ejecucion Taller 


- instrucciones ejecutar el taller SQL:
  ```
  1. abrir el achivo main.py
  2. seccion superior derecha buscar el icono de play o ejecutar ▶️
  3. seleccionar el entorno de ejecucion
  4. validar la respuesta sobre el terminal.
  ```
